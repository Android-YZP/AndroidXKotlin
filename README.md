# AndroidXKotlin
##   以下符号会始终解释为关键字，不能用作标识符：
   
       as用于类型转换为导入指定一个别名
   
       as? 用于安全类型转换
   
       break 终止循环的执行
   
       class 声明一个类
   
       continue 继续最近层循环的下一步
   
       do 开始一个 do/while 循环（后置条件的循环）
   
       else 定义一个 if 表达式条件为 false 时执行的分支
   
       false 指定布尔类型的“假”值
   
       for 开始一个 for 循环
   
       fun 声明一个函数
   
       if 开始一个 if 表达式
